اختبار الدقائق الثلاث

هل الإضافة ظاهرة؟

ابحث في Extensions عن INI Brain، ثم تأكد أنها Enabled. أعد تحميل VS Code.

هل عقل المشروع موجود؟

اضغط Scan Project، ثم تحقق من وجود .brain/metadata.json وAGENTS.md.

هل MCP يعمل؟

في الوكيل اطلب ini_brain_status. إذا لم تظهر الأداة، المشكلة في إعداد العميل أو إعادة التشغيل.

هل المسار صحيح؟

اقرأ قيمة workspace في النتيجة. يجب أن تكون مجلد مشروعك، لا مجلد الإضافة.

المشاكل الشائعة

المشكلةالسبب المعتادالحل
Codex لا يرى أدوات INI Brainلم يُحدّث config أو لم يُعد تشغيلهشغّل installer، افحص ~/.codex/config.toml، أغلق Codex وافتح مهمة جديدة.
Cline يظهر الخادم Offlineمسار server.js خاطئ أو dist غير مبنيشغّل npm run compile في مجلد INI Brain، ثم اضغط Install MCP وأعد تحميل VS Code.
Claude Desktop لا يتصلالتطبيق بقي في system trayQuit كاملًا، افحص JSON، ثم افتحه من جديد.
workspace مشروع خاطئالعميل شغّل MCP من مجلد مركزيافتح المهمة من مجلد المشروع، أو مرّر workspace في الأداة.
Smart Setup لا يكتشف واجهة في مجلد فارغلا توجد ملفات ولا وصفاكتب React/واجهة/موقع داخل project_request.md ثم أعد Preview.
لا توجد حزمة Code Intelligence متقدمةالمحرك اختياريLite Graph يعمل تلقائيًا. للتثبيت الصريح استخدم -InstallCodeIntel.
التوكينات زادت في سؤال بسيطاستدعيت سياق المشروع بلا حاجةلا تستخدم Auto Brief للأسئلة العامة. استخدم budgetChars أصغر للمهام الصغيرة.
الإجابة قصيرة لدرجة غير مفهومةCaveman أو Ponytail مستوى شديدSwitch Level إلى Lite، أو Disable All، ثم أعد المهمة.
التحديث لم ينزّل repo كاملًاهذا السلوك المقصودINI Brain يجلب الملفات المسجلة فقط. الـarchive الكامل للنسخ الاحتياطي وليس لكل مشروع.

أوامر فحص للمطور

node --version
npm --version
npm run compile
npm run smoke:all
npm run audit:public
npm run package

اختبار الخادم يدويًا:

node dist/mcp/server.js
ملاحظةالخادم ينتظر JSON-RPC على stdin؛ ظهوره بلا واجهة رسومية طبيعي. الاختبار الأفضل هو npm run smoke:mcp.

قبل أن تحذف شيئًا

  • لا تحذف .brain/backups إذا كنت تحتاج الاستعادة.
  • استخدم Disable/Remove All بدل حذف أجزاء عشوائية.
  • احفظ نسخة من قراراتك المهمة.
  • لا تستخدم git reset --hard لحل مشكلة إعداد.
مفتاح APIلا تكتبه في README أو AGENTS.md أو project_request.md أو memories.json. أدخله من Settings ليُحفظ في SecretStorage.
