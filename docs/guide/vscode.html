الأزرار الأساسية

الزرمتى أستخدمه؟ماذا يفعل؟هل يغيّر ملفات؟
Scan Projectأول مرة في مشروع موجوديفحص الملفات محليًا وينشئ عقل المشروع وملفات الوكلاء.نعم: .brain وAGENTS.md ومرايا المهارات.
Smart Setupمشروع جديد أو حين تريد أفضل إعداديفهم اللغات ووصف المشروع والوكلاء، ثم يعرض حزمًا مقترحة.المعاينة لا. التطبيق بعد موافقتك نعم.
Rebuild Brainبعد تغييرات كبيرة أو إذا أصبح الملخص قديمًايعيد الفحص وبناء خرائط السياق.يحدّث ملفات INI Brain فقط.
Guided Setupعندما يكون المجلد فارغًاينشئ project_request.md لتكتب فيه فكرتك.ينشئ ملف وصف المشروع.
Ask AIلسؤال المزود الاختياري من داخل VS Codeيرسل السؤال مع سياق المشروع المركز إلى المزود الذي أعددته.لا، إلا إذا اخترت Auto Mode لاحقًا.
Auto Modeعندما تريد من AI اقتراح تعديلات ملفاتيعرض الخطة والتغييرات، يطلب التأكيد، ثم يحفظ نسخة احتياطية قبل الكتابة.نعم بعد موافقة صريحة.
Generate Projectلبناء مشروع جديد من الوصفيستخدم المزود لإعداد خطة ملفات مشروع.نعم بعد المعاينة والتأكيد.

الفهم والذاكرة والجودة

الزروظيفته بلغة بسيطة
Agent Guideيعيد إنشاء AGENTS.md وتعليمات المشروع للوكلاء.
Skills & Workflowينشئ المهارات المناسبة للغات وأطر المشروع ومسار العمل.
Onboardingيكتب دليل «ابدأ من هنا» يشرح أهم الملفات ونقاط الدخول.
Explain Fileافتح ملفًا في المحرر، ثم اضغطه لمعرفة exports والتبعيات ومن يعتمد عليه.
Analyze Impactيفحص ملفات Git المتغيرة ويخبرك بما قد يتأثر.
Quality Guardsينشئ حراس Clean Code والاختبارات والواجهة وإرشادات التغيير البسيط.
Save Memoryيحفظ قرارًا أو خطأً أو تفضيلًا ليسترجعه الوكلاء لاحقًا.
Search Memoryيبحث في الذاكرة المحلية بكلماتك.
Memory Profileيعرض عدد الذكريات وأهم المفاهيم والملفات المتكررة.
Restore Backupيعيد ملفًا من نسخة Auto Mode الاحتياطية. راجع التاريخ قبل الاختيار.

الاتصال بالوكلاء والإعدادات

Copy for Cline

ينسخ سياقًا مركزًا تلصقه يدويًا في Cline. مفيد حتى قبل إعداد MCP.

Copy MCP Config

ينسخ JSON العام لخادم INI Brain. استخدمه في أي عميل MCP يدعم stdio.

Install MCP

يثبت إعداد MCP تلقائيًا داخل Cline على Windows ويحافظ على الخوادم الموجودة.

Settings

يفتح إعدادات المزود والنموذج والمهلة والتأكيد والنسخ الاحتياطية. مفتاح API يُحفظ في SecretStorage.

Visual Guide

يفتح هذا الدليل المصور من داخل الإضافة.

Refresh / Clear Console

Refresh يعيد رسم اللوحة، وClear Console يمسح رسائل اللوحة فقط ولا يحذف ملفات أو ذاكرة.

بطاقات المميزات

البطاقة والزرالنتيجة
Token Savings → Enable Allيفعّل Caveman وPonytail وClaude Lean. للمبتدئ نوصي Smart Setup لأنه يختار Lite المتوازن.
Disable All / Dashboard / Switch Levelإلغاء التوفير، عرض السجل المقاس، أو اختيار Lite/Full/Ultra. ابدأ بـLite.
Knowledge Graph → Build / Viewيبني رسمًا محليًا للملفات وعلاقات الاستيراد ثم يعرضه.
Knowledge Graph → Find Impactافتح ملفًا ثم اعرف من يعتمد عليه مباشرة وبشكل متسلسل.
Spec-Driven Dev → New Specيكتب ماذا نبني ولماذا وقصص المستخدم ومعايير القبول.
Plan → Tasks → Implementيحوّل المواصفة إلى معمارية، ثم مهام صغيرة، ثم يفتح المهمة التالية.
Engineering Workflowيضيف اكتشاف المنتج ومراجعة الخطة والتحقيق والأمن ومراجعة الإصدار، أو يزيلها.
Adapters → Install Allينشر المهارات المفعلة إلى الوكلاء المكتشفين فقط.
Adapters → Remove Allيزيل مرايا مهارات INI Brain من الوكلاء بعد تأكيدك.
Upstream Updates → Check Nowيفحص commit metadata صغيرة عند طلبك، ثم يعرض التحديثات قبل التطبيق.
Upstream Updates → Configureيحفظ تفضيل الفحص الدوري. التطبيق يظل بحاجة موافقة.
