# دورة حياة الذاكرة

الإصدار 3.1.0 يضيف طبقة أقوى لإدارة ذاكرة INI Brain المحلية، مستلهماً بعض أفكار AgentMemory بدون إضافة خادم خارجي أو قاعدة بيانات.

## الحقول الجديدة

يمكن لكل ذاكرة أن تحتوي على:

- `confidence`: رقم من 0 إلى 1 يساعد في ترتيب نتائج البحث.
- `expiresAt`: وقت اختياري لانتهاء صلاحية الذكريات المؤقتة.
- `pinned`: يمنع حذف الذاكرة أثناء التنظيف.
- `origin`: وصف قصير لمصدر الذاكرة.

الذكريات القديمة تظل قابلة للقراءة، وأي حقول ناقصة تحصل على قيم افتراضية آمنة عند القراءة.

## التنظيف الآمن

أداة `ini_brain_memory_compact` تعرض معاينة جافة افتراضياً قبل أي كتابة. يمكنها:

- حذف الذكريات المنتهية فقط إذا لم تكن مثبتة.
- دمج الذكريات المتشابهة فقط عندما يكون النوع متطابقاً والمفاهيم متداخلة.
- الحفاظ على الذكريات المثبتة حتى لو كانت قديمة أو منتهية.

استخدم `apply=true` فقط عندما تريد كتابة نتيجة المعاينة إلى `.brain/memories.json`.

## الإحصاءات

أداة `ini_brain_memory_stats` تعرض العدد الكلي، عدد الذكريات المثبتة، المنتهية، توزيع الأنواع، متوسط الأهمية، ومتوسط الثقة.

## الخصوصية

لا تحتاج الميزة إلى خادم ذاكرة خارجي، ولا يتم تضمين AgentMemory كاعتماد runtime داخل الإضافة.
