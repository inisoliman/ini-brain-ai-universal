# Knowledge Graph - Project Dependency Map

## Benefit
- Visual project understanding.
- Blast radius for any file change.
- Much lower-token project queries.

## Enable
- `INI Brain: Build Knowledge Graph` - build it.
- `INI Brain: Show Knowledge Graph` - render Mermaid.
- `INI Brain: Find Impact` - inspect the active file.

## Classification
Every edge is tagged as `EXTRACTED`, `INFERRED`, or `AMBIGUOUS`.
