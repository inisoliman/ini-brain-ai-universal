# Ponytail - Lazy Code Writer

## Benefit
About 54% savings in generated code.

## Philosophy
1. Does it need to exist? (YAGNI)
2. Standard library?
3. Native platform feature?
4. Existing dependency?
5. One line?
6. Minimum working code.

## Enable
`INI Brain: Enable Ponytail Mode`.

## Modes
lite / full / ultra.

## Commands
- `/ponytail-review` - review the file.
- `/ponytail-audit` - review the project.
- `/ponytail-debt` - find debt.
- `/ponytail-gain` - pre-feature checklist.

## Never Skimp On
Validation, error handling, security, and accessibility.
