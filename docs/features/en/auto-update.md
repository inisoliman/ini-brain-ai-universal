# Auto-Update

## What It Does
Checks 7 public GitHub repositories weekly and reports upstream changes.

## Sources
- ponytail
- caveman
- claude-token-efficient
- spec-kit
- superpowers
- graphify
- token-optimizer

## Settings
- `iniBrain.autoUpdate.enabled` (default: false)
- `iniBrain.autoUpdate.intervalHours` (default: 168)
- `iniBrain.autoUpdate.requireApproval` (default: true)

## Commands
- `INI Brain: Check Upstream Updates`
- `INI Brain: Configure Auto-Update`

No API key required for public GitHub API checks.
