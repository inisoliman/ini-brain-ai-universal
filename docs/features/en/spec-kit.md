# Spec-Kit Lite - Spec-Driven Development

## What Is It?
A local TypeScript version of Spec-Driven Development without Python.

## Workflow
1. `INI Brain: Create Spec` - write the what.
2. `INI Brain: Generate Plan` - add the stack.
3. `INI Brain: Break into Tasks` - split the work.
4. `INI Brain: Open Next Task` - implement.

## Files
- `.specify/memory/constitution.md`
- `.specify/specs/<slug>/spec.md`
- `.specify/specs/<slug>/plan.md`
- `.specify/specs/<slug>/tasks.md`
